# GraphQL-FullStack

## To get started:

* _Navigate_ to the `server` folder.
* Run `npm install`.
* Run `npm start`.

This will start the **GraphQL API** server.

In _another_ Terminal window,

* Navigate to the `client` folder.
* Run `npm install`.
* Run `npm start`.

This will open up `localhost:3000` in your web browser.
